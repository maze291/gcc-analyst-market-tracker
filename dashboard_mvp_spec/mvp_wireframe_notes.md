# MVP Wireframe Notes

## First View

- Country selector: UAE, Saudi Arabia, or both where coverage passes.
- KPI strip: total analyst postings, new postings this period, salary coverage rate, restricted-source share.
- Trend chart: weekly or monthly posting counts.

## Main Analysis

- City distribution chart.
- Analyst role mix chart.
- Top skills aggregate count table.
- Employment type and work arrangement distributions.

## Compliance Footer

- State that the dashboard shows aggregate statistics only.
- State that no full job postings, recruiter data, or direct application links are displayed.
- State provider/source limitations where required by written permission.
