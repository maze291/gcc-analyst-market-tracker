# Feasibility Tests

This folder contains Week 0 sample queries, manual review templates, scoring rules, and result templates.

## Workflow

1. Run the first-pass query set before the full grid.
2. Pull tiny samples locally only.
3. Normalize into aggregate-safe fields.
4. Review 50-100 postings per active API where coverage allows.
5. Apply compliance gate before weighted scoring.

## Active APIs

- JSearch / OpenWeb Ninja
- Careerjet
- Jooble

Adzuna is documentation-only unless written permission is obtained.
