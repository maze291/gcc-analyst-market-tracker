# Jooble Source Rule

## Week 0 Role

Secondary validation source.

## Production Status

Validation only until terms clarify storage, derived analytics, and public aggregate reporting.

## Use Constraints

- Do not treat Jooble as the legal foundation for a public job database.
- Do not publish Jooble listing text.
- Store source domain only.
- Extract skills temporarily from snippets/descriptions, then discard source text.
- Flag restricted source domains.

## Known Risk

Jooble's API page supports posting job results on a website, but general terms limit copying/republication and place responsibility on the user for third-party content and terms.
