# Restricted Sources

Any posting whose source or apply domain is in this list must be flagged and excluded from public source-link display.

## Blocked / High-Risk Domains

- linkedin.com
- indeed.com
- glassdoor.com
- bayt.com
- naukrigulf.com
- gulftalent.com

## Handling Rule

- Store only the registered source domain, not the full URL.
- Do not display direct links to these domains publicly.
- Include them only in aggregate source-mix statistics when doing so does not identify individual postings.
- Add newly discovered high-risk publisher domains during review.
